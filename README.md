# Valorant Tools

Free Valorant sensitivity converter and crosshair generator.

## Features

### Sensitivity Converter
- Convert sensitivity between 12+ games (Valorant, CS2, Apex, OW2, Fortnite, etc.)
- Calculate cm/360° and eDPI
- Quick reference for all games at once

### Crosshair Generator
- Real-time crosshair preview
- 8 colors available
- Export Valorant crosshair code
- Pro player presets (TenZ, Faker, etc.)

## Tech Stack

- React + Vite
- Tailwind CSS
- LocalStorage for data persistence

## Development

```bash
npm install
npm run dev
```

## Build

npm run build
